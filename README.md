# Sentiment Analysis Basic

A simple Machine Learning project that analyzes text and predicts whether the sentiment is positive or negative.

## Features
- Train a sentiment classification model
- Predict sentiment from text input
- Simple Python implementation using machine learning

## Project Structure

train.py – trains the machine learning model  
predict.py – predicts sentiment from text input  
app.py – simple application interface  
vectorizer.pkl – saved text vectorizer  
sentiment_model.pkl – trained model  
requirements.txt – project dependencies

## Installation

Clone the repository

pip install -r requirements.txt

## Usage

Run the application

python app.py

Enter text and the model will predict the sentiment.

## Technologies Used

- Python
- scikit-learn
- NLP (Natural Language Processing)
